# garage-door-controller
A garage door controller using MicroPython

## Getting Started

This project includes a simple Hello World MicroPython script to get started.

### Running the Hello World Script

To run the hello world script:

```bash
python main.py
```

Or on a MicroPython-enabled device:

```bash
micropython main.py
```

## Files

- `main.py` - Hello World MicroPython script for the garage door controller
